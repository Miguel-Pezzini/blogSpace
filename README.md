# Astronomy Blog Project in Vue

This project aims to create an interactive and responsive blog developed in **Vue.js** and **Vuetify**, focused on Astronomy.

## Features

- **Content Publishing**: Allows the creation of posts covering a variety of topics, including:
  - Opinions on astronomical discoveries
  - Curiosities about the universe
  - Recent research and developments in the field

- **News Fetching**: Integration with the public **NewsAPI** to retrieve the latest news on Astronomy, ensuring users have access to the most up-to-date information.

- **User-Friendly Interface**: Utilizes Vuetify for a modern and responsive user experience, making navigation easy on both mobile and desktop devices.

- **Comments and Interaction**: (if applicable) Implementation of a commenting system for readers to interact and share their thoughts.

## Technologies Used

- **Vue.js**: A progressive framework for building user interfaces.
- **Vuetify**: A material design component library for Vue.js.
- **NewsAPI**: An API that provides access to news from various sources about Astronomy.
